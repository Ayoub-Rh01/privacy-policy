Politique de confidentialité

Nous attachons une grande importance à la protection de vos données personnelles. Cette politique explique quelles données nous collectons, comment nous les utilisons, et quels sont vos droits.

1. Données collectées

Nous ne collectons pas de données personnelles directement à l’exception des tokens d’accès Meta nécessaires pour utiliser l’API Instagram. Ces tokens nous permettent d’agir en votre nom pour publier des contenus et lire des informations liées à votre compte Instagram.

2. Utilisation des données

Les tokens d’accès sont utilisés uniquement pour interagir avec Instagram via l’API officielle :

  • Publication automatique de posts,
  • Lecture d’informations de votre compte.

Les données récupérées via l’API peuvent être traitées par un agent d’intelligence artificielle tiers (OpenAI/ChatGPT) pour automatiser certaines tâches.

3. Partage des données

Vos tokens d’accès sont stockés uniquement dans notre environnement sécurisé (n8n). Les informations traitées peuvent être transmises à OpenAI pour les besoins du traitement, conformément à leurs politiques de confidentialité. Aucune autre donnée n’est partagée avec des tiers.

4. Sécurité

Nous mettons en place des mesures de sécurité pour protéger vos données :

  • Accès restreint au système n8n,
  • Transfert sécurisé des données vers les services tiers,
  • Suppression régulière des données après traitement.

5. Conservation des données

Les tokens et données associées sont conservés pendant une durée maximale de 6 mois, après quoi ils sont supprimés ou rendus inutilisables.

6. Vos droits

Vous pouvez à tout moment exercer vos droits d’accès, de modification, de suppression de vos données, ou retirer l’accès de l’application via les paramètres de votre compte Instagram. Pour toute demande, vous pouvez nous contacter à l’adresse : renovation.rh@outlook.fr.
